# 港翼科技 GONYIK 官网 — 项目日志

## 项目概述
港翼科技（GONYIK）科技面料品牌官网，面向 B2B 与 B2C 双端用户。全站内容通过内置 CMS 维护，零代码更新。

---

## 技术栈
- **前端**：React 19 + TypeScript + Vite + Tailwind CSS + Framer Motion
- **后端**：Express + JSON 文件数据库（db.json）
- **字体**：Inter（英文）+ 阿里巴巴普惠体（中文）

## 关键设计决策

### 1. CMS 方案：JSON 文件数据库
- 选择 Express + JSON 文件（db.json）而非 SQLite/MySQL
- 原因：零配置、单文件、部署最简单、未来迁移云服务器方便
- 首次启动自动注入默认数据

### 2. 默认数据策略
- 未配置时自动生成默认内容（Logo、背景图、产品数据等）
- 后台修改后立即生效，无需重启

### 3. 字体选择
- 英文：Inter（Google Fonts CDN）
- 中文：阿里巴巴普惠体（jsdelivr CDN）
- 不影响项目包体积，不阻塞渲染

### 4. 响应式断点
- 桌面大屏：≥1440px
- 桌面：1024–1439px
- 平板：768–1023px
- 移动端：<768px（iPhone 375px 为标准适配）

## 启动方式
```bash
# 开发模式（前后端同时启动）
npm run dev

# 或分别启动
npm run server   # 后端 localhost:3001
npm run client   # 前端 localhost:5173

# 生产构建
npm run build

# Windows 一键启动（双击）
start.vbs
```
